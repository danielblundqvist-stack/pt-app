# PT-app för webben

Det här är en enkel privat PT-app byggd med React + Vite.

## Kör lokalt
1. Installera Node.js
2. Kör:
   npm install
   npm run dev

## Deploy till Vercel
1. Packa upp zip-filen
2. Gå till Vercel
3. Skapa nytt projekt
4. Importera mappen
5. Klicka Deploy

Vercel känner normalt av Vite automatiskt.
Build command: npm run build
Output directory: dist
